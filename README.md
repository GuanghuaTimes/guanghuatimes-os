### how to upload to Alibaba Cloud 2024

https://docs.serverless-devs.com/getting-started/

1. install serverless devs

```bash
npm install @serverless-devs/s -g
```

2. configuration of this tool

```bash
s config add
```

3. deploy

```bash
s deploy -y
```
